# mont-validators-workspace

Workspace для библиотеки **@mont/validators** — Angular-валидаторов и декораторов для Reactive Forms.

## Установка библиотеки

```bash
npm install @mont/validators
```

## Использование

См. [projects/mont-validators/README.md](projects/mont-validators/README.md).

## Сборка и тесты

```bash
ng build mont-validators
ng test mont-validators
npm run test:coverage
```
